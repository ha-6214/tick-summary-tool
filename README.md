# tick-summary-tool

歩み値CSVを端末内で集計し、分析用のMDファイルを出力するツールです。
データは外部に送信されず、すべてブラウザの中だけで処理されます。

## 使い方

1. `https://ha-6214.github.io/tick-summary-tool/` を開きます
2. 歩み値CSVを選ぶ
3. 銘柄情報を貼り付ける
4. 前日の集計結果（.md）を選ぶ（省略可）
5. 「集計する」を押し、結果をコピーまたはMDで保存する

## 銘柄情報のテンプレート

```
日付: 2026年7月22日
銘柄コード: 5802
銘柄名称: 住友電工
年初来高値: 3019
年初来高値の日付: 2026年6月30日
信用売れ残: 978500
信用買い残: 11451200
信用倍率: 11.70
```

項目の順番は自由です。空行と `#` で始まる行は読み飛ばします。
上記以外の項目（取得時刻など）は使わず、画面に「使わなかった項目」として表示します。
